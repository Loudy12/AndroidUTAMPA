# AndroidUTAMPA
2024-2025 Capstone Project. 
A campus app for the University of Tampa. 
AndroidUTAMPA is a collaborative mobile application developed for the University of Tampa to improve the student campus experience.  
The app provides real-time and centralized access to campus resources, navigation tools, and parking availability information.

## Overview
This project was built by a diverse student team to design and implement a fully functioning university app.  
The application focuses on improving day-to-day student life by making key campus information easily accessible from a single mobile platform.

I led a small development group responsible for designing and implementing the Android interface and ensuring cohesive integration with the broader project.

## Features
- **Parking Garage Capacity Tracker**  
  Displays parking availability to help students quickly locate open spaces.
  
- **Interactive Campus Map**  
  Provides visual navigation across campus locations and facilities.
  
- **Centralized Campus Resource Hub**  
  Offers quick access to important student resources, services, and information.
  
- **Multi-screen Navigation & UI**  
  Structured user flow for intuitive interaction across features.
